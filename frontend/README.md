## 🌟 About The Project

This repository contains the source code for my personal developer portfolio. Built with a focus on cutting-edge aesthetics and robust performance, the application leverages responsive layouts, interactive glassmorphism, dynamic animations, and seamless scrolling to showcase my skills as a Full-Stack Software Engineer. 

The site serves as a central hub for my professional journey, detailing my technical competencies, my educational progress, and my featured deployments like the **EDOT Platform** and **ClientFlow CRM**.

### 💼 Key Features
* **Permanent Dark Mode Aesthetics:** Deep navy/cyan palettes utilizing `backdrop-blur` UI modules.
* **Complex Data Display:** Clean, timeline-based Experience & Education sections alternating structurally down the page.
* **Animated Skill Visualizations:** Features an infinite-scroll marquee and a categorized, glowing icon grid.
* **ScrollSpy Navigation:** Interactive navigation menu that actively highlights the user's current section as they scroll.
* **Direct Communication System:** Integrated Web Contact form targeting standard email ingestion along with Google Maps structural embed.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need `npm` (Node Package Manager) installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/kenenisabeyan/myPortifolio.git
   ```
2. Navigate to the project directory
   ```sh
   cd myPortifolio
   ```
3. Install NPM packages dependencies
   ```sh
   npm install
   ```
4. Run the local development server (Vite)
   ```sh
   npm run dev
   ```
5. Build for production (Optional)
   ```sh
   npm run build
   ```

## 📂 Project Structure

```text
├── public/                 # Static assets (images, pdfs)
├── src/
│   ├── components/         # React independent UI sections (Hero, Work, Contact, etc.)
```
